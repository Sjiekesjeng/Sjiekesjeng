## Featured Projects

### [Map-Extractor](https://github.com/Sjiekesjeng/Map-Extractor)

[![Map-Extractor Picture](https://iili.io/qniHefe.md.png)](https://freeimage.host/i/qniHefe)

> **Map-Extractor** is a script built in **Python** (written for use in **QGIS**) that converts georeferenced raster map images into clean, analysis-ready raster and vector outputs by identifying defined classification colours, removing map overlays (roads, text, grid lines), and exporting each colour class as both GeoTIFF and GeoPackage files. This project demonstrates my ability to gather and modify data intelligently and pragmatically. You can check out the repository [here](https://github.com/Sjiekesjeng/Map-Extractor).

### [WEEB-bot](https://github.com/Sjiekesjeng/WEEB-bot)

[![WEEB-bot Picture](https://iili.io/qnsRb1e.md.png)](https://freeimage.host/i/qnsRb1e)

> **WEEB-bot** is a program that automates the process of finding out whether a new episode has come out for a series you are watching, and downloading it through the use of P2P torrent networking. The program is built in **Python** and makes use of **QBittorrent**. This project demonstrates my ability to recognize patterns and automate everyday tasks. You can check out the repository [here](https://github.com/Sjiekesjeng/WEEB-bot).

### [CamDar](https://www.linkedin.com/posts/cauberghuygen_stagiair-baanbrekend-camdar-activity-6856575729526718464-EWhz?utm_source=share&utm_medium=member_desktop&rcm=ACoAAC5h7LcB6Me6qLfZSfzcfF5xycuC05weogk)

[![CamDar Picture](https://iili.io/qnsMXOF.md.jpg)](https://freeimage.host/i/qnsMXOF)

> **CamDar** is a custom-built measuring tool that allows for automatic determination of train model and velocity, built with a microcomputer (Raspberry Pi), camera and radar. This project showcases my skills in distilling a program of requirements, researching methods, designing and building prototypes, programming in Python and pragmatic troubleshooting. The code I wrote to realise this is intellectual property of [Cauberg Huygen](https://cauberghuygen.nl/en/) and not publically available, but I would gladly discuss the process and details in person.
